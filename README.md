# Individual-Git-Exercise
A Simple Calculator app with Two Features:
1. Addition
2. Subtracion

### Addition

this Feature lets you add multiple integers inputted in one line with spaces in between
Example input : 1 2 3 4 5
output: Sum =15

Note: Any other character other than space between the integers will result in error

### Division

This Feature lets you divide two integers
Example input: Numerator:10
Denominator: 5
output:10/5 = 2.0 

Note Division with 0 is not possible  and will result in error
